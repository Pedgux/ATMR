## Info

Inspired by Nathan Daniels' paper: [Roguelike Theory of Relativity (RTOR) ](docs/Nathan_Daniels_-_RTOR__Roguelike_Theory_of_Relativity_(2009).pdf)  
Built with Spectre.Console for console UI and rendering: https://spectreconsole.net/  
Uses Arch as it's ECS: https://github.com/genaray/Arch  
Trello page: https://trello.com/b/mGoWnU28/atmr  

This is a personal project, which I hope to bring to light someday. 

## Installation

Prerequisites: .NET 10 SDK installed (https://dotnet.microsoft.com/download).

Quick start:

```bash
git clone https://github.com/Pedgux/ATMR.git
git submodule update --init --recursive
cd ATMR
dotnet run --project src/ATMR.csproj
```

Or run directly from the `src` folder:

```bash
cd src
dotnet run
```
